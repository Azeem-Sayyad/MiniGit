Need Help?

Everything you need to know to use MiniGit smoothly

MiniGit is a lightweight repository management system designed for students and small teams to manage projects easily without complex Git commands.
Registration is required to create repositories, add commits, and view activity logs. Demo and features can be viewed without registering.
After logging in, go to the dashboard and click on "Create Repository". Enter repository details and start managing your project.
A commit represents a change or update made to a repository. It helps track progress and maintain version history.
Currently, password recovery is handled by the admin. Please contact support for assistance.
